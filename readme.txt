Habanero Testability v1.2, 3 December 2010

This package contains the source of Habanero Testability, a library to enable easier testing of Habanero business objects. It is used extensively by the generated tests created by FireStarter

It includes the following:

- Habanero.Testability.dll
- Habanero.Testability.Helpers.dll

This version of Habanero Testability was built against Habanero Core 2.5.1 and Smooth Habanero 1.4
